Dark Scan

Dark Scan is a port scanner that uses the Tor network to scan targets. To connect to the Tor network Dark Scan uses a pre-compiled standalone Tor binary, there is no need to install/configure anything on the host system. The connection to the Tor network is done automatically without user interaction.
